Keamanan Data

Kami memprioritaskan keamanan data Anda. Semua request yang dikirimkan melalui XTE APIs dilindungi oleh enkripsi SSL/TLS tingkat tinggi untuk mencegah penyadapan data oleh pihak yang tidak berwenang.

Data yang Dikumpulkan

XTE hanya mengumpulkan data minimal yang diperlukan untuk fungsionalitas layanan, seperti:

  • check Log request untuk keperluan statistik
  • check IP Address untuk proteksi rate-limit
  • check Metadata API untuk perbaikan bug

Penggunaan Informasi

Informasi yang kami kumpulkan murni digunakan untuk meningkatkan kualitas layanan XTE APIs. Kami menjamin bahwa kami tidak akan pernah menjual atau memberikan data personal Anda kepada pihak ketiga untuk kepentingan iklan.

Penyimpanan Data

Log aktivitas API secara otomatis akan dihapus secara berkala setelah jangka waktu tertentu untuk memastikan privasi pengguna tetap terjaga dan database tetap efisien.
