TERMS OF SERVICE.

Terakhir diperbarui: 16 Februari 2026

01. Penggunaan Layanan

Dengan mengakses XTE APIs, Anda setuju untuk mematuhi semua kebijakan yang berlaku. Layanan ini disediakan untuk membantu pengembangan perangkat lunak secara legal dan bertanggung jawab.

02. Batasan Akses

Dilarang keras melakukan aktivitas spamming, brute force, atau mencoba merusak infrastruktur server XTE. Kami berhak mencabut akses API key Anda tanpa pemberitahuan jika terdeteksi aktivitas mencurigakan.

03. Tanggung Jawab

XTE APIs tidak bertanggung jawab atas penyalahgunaan data hasil API oleh pihak ketiga. Pengembang bertanggung jawab penuh atas implementasi API di aplikasi masing-masing.

04. Perubahan Layanan

Kami dapat menambah, mengubah, atau menghentikan endpoint API tertentu untuk keperluan maintenance atau peningkatan sistem demi performa yang lebih stabil.
